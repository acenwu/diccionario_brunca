# Instrucciones de ejecución de Wireframe

## Instalar dependencias

Primero se deben de instalar las dependencias requeridas con el siguiente comando:

`npm install`

## Ejecución

Una vez instaladas las dependencias se ejecuta el Wireframe con el siguiente comando:

`npm run dev`

## Acceso

Para acceder al Wireframe se debe abrir un navegador y dirigirse a la dirección indicada por el programa en terminal, por ejemplo:

`http://localhost:5173/`
